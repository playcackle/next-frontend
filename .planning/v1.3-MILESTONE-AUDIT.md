## Tech Debt

### Phase 10: Sentry Foundation
- **OBS-05 phase context gap:** `setSentryGameContext(gameroom.game_ws_url)` in `page.tsx:110` does not pass the current game phase — `gameroom.phase` in Sentry events is always `"unknown"`. Fix: read `gamePhaseAtom` and pass it as second argument.

### Phase 12: Performance Baselines
- **Dev-only WebVitalsLogger:** `WebVitalsLogger` is gated to `NODE_ENV=development`. No production runtime measurement path for LCP, CLS, INP. The Phase 13 LCP fix cannot be confirmed in production without enabling real-user measurement (e.g., Vercel Speed Insights or web-vitals library reporting endpoint).
- **SUMMARY documentation gap:** `12-02-SUMMARY.md` lists `wdyr.ts` and `WdyrInit.tsx` as `key-files.created` but both files were removed from codebase. The SUMMARY does not reflect final state accurately.

### Phase 13: Performance Fixes
- **Production LCP unverifiable:** The LCP improvement from the `INITIAL_SESSION` guard was accepted as code-correct, but no production measurement was taken. The baseline (4324ms) was from dev; the fix prevents a second render pass that was visible in dev. Recommend adding `web-vitals` RUM reporting in a future observability phase.
